║                      TROUBLESHOOTING                              ║
╚══════════════════════════════════════════════════════════════════╝

❌ Connection Refused?
   → Verwende: http://host.docker.internal:8080/v1
   → NICHT: http://localhost:8080/v1

❌ Unauthorized (401)?
   → API Key muss mit "Bearer " beginnen (mit Leerzeichen!)
   → Prüfe Copy & Paste - keine zusätzlichen Zeichen

❌ No Models?
   → Warte 30 Sekunden nach Save
   → Refresh die Seite
   → Prüfe: http://localhost:8080/v1/models im Browser

❌ Token expired? (nach 1 Stunde)
   → Generiere neuen Token:
     curl -X POST http://localhost:8080/oauth/token \
       -H "Content-Type: application/json" \
       -d '{
         "grant_type": "client_credentials",
         "client_id": "openwebui-production",
         "client_secret": "openwebui-secret-2026-secure-key-abc123xyz789",
         "scope": "read write"
       }'
