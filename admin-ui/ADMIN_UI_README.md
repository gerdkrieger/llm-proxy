# LLM-Proxy Admin UI

Modern web interface for LLM-Proxy administration built with Svelte + TailwindCSS.

## Features
- Dashboard with metrics
- OAuth Client Management (CRUD)
- Cache Management
- Usage Statistics
- Provider Status

## Quick Start
```bash
npm install
npm run dev
```

Access at: http://localhost:5173

Default API Key: `admin_dev_key_12345678901234567890123456789012`

## Tech Stack
- Svelte 5
- Vite
- TailwindCSS 4
- Svelte Routing
