🛡️ Content Filter Management (Advanced)

Bulk-Import von Filtern über Textarea oder CSV-Upload

➕ Bulk Import per Textarea

📋 Format:

Jede Zeile ist ein Filter. Verwende eines dieser Formate:

  • Komma-separiert: pattern, replacement, type, priority, description
  • Tab-separiert (TSV): pattern replacement type priority description
  • Einfach (nur Wörter): Ein Wort pro Zeile wird automatisch als Word-Filter mit [FILTERED] ersetzt

Beispiel:

badword, [FILTERED], word, 100, Filtert Schimpfwörter confidential information, [REDACTED], phrase, 95, Vertrauliche Infos \b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b, [EMAIL], regex, 90, Email-Adressen

📄 CSV-Datei Upload

📋 CSV Format:

Die CSV-Datei sollte folgende Spalten haben (Header-Zeile ist optional):

  • pattern - Das zu filternde Pattern (erforderlich)
  • replacement - Der Ersetzungstext (erforderlich)
  • filter_type - word, phrase oder regex (optional, Standard: word)
  • priority - Priorität 0-999 (optional, Standard: 100)
  • description - Beschreibung (optional)
  • case_sensitive - true/false (optional, Standard: false)
  • enabled - true/false (optional, Standard: true)

Beispiel CSV:

pattern,replacement,filter_type,priority,description badword,[FILTERED],word,100,Offensive Sprache damn,[*],word,100,Kraftausdrücke confidential information,[REDACTED],phrase,95,Vertraulich
