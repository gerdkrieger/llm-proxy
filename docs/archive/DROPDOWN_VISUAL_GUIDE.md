3. Navigate to "рҹ”җ Security & Credentials"

7. Select type: `regex`
8. Click "Create"
9. вң… Filter created with `[***AWS_KEY***]` replacement

6. Click "Create"
7. вң… Filter created with `[COMPANY_SECRET]` replacement

**Flexibility:** Still allows custom values for unique cases

1. Find filter with replacement `[CREDIT_CARD]`

6. Type: `[***PAYMENT_INFO***]`
7. Click "Update"
8. вң… Filter updated to custom replacement

**Smart detection:** Knows current value is a template

7. вң… Filter updated to template replacement

**Upgrade path:** Easy to switch from custom to template
